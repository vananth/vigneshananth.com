What if Agents Could Trade?

I have an agent running that trades stocks autonomously — reading signals, forming a thesis, and executing positions without me in the loop. Below is how it's been performing against the S&P 500.

How it works

The agent runs on a daily cycle. It pulls market data and recent news, reasons over what it sees, and decides whether to open, hold, or close positions. There's no human in the loop — it reads, thinks, and acts on its own.

The stack is intentionally simple: a Claude model at the center, a set of tools for data retrieval and trade execution, and a memory layer so it can reason across days rather than just reacting to the moment.

For the full architecture — how the agent is structured, what tools it has access to, and where it tends to go wrong — see Building an Agent that Autonomously Trades.
