What if Agents Could Trade?

An experiment in agentic stock trading. I gave an AI agent the tools, context, and autonomy to make real market decisions. It hasn't lost me all my money yet. It reads signals, forms a thesis, and executes positions without me in the loop.

How it works

The agent runs on a daily cycle. It pulls market data and recent news, reasons over what it sees, and decides whether to open, hold, or close positions.

The stack is built around an LLM at the center, a set of tools for data retrieval and trade execution, and a memory layer so it can reason across days and react to past decisions, rather than just reacting to the moment.

For the full architecture, how the agent is structured, what tools it has access to, and what the memory layer looks like read on here — Building an Agent that Autonomously Trades.
